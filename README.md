# 表格转换工具

一个运行在浏览器中的本地表格转换工具。双击 HTML 文件即可使用，无需安装，所有数据在本地处理，不会上传到任何服务器。

## 功能

### 表格 → 纯文本

将 Excel（.xlsx/.xls）或 CSV 文件转换为纯文本格式，支持三种输出：

- **CSV** — 逗号分隔值，兼容各种表格软件
- **Markdown 表格** — 适合文档编写和代码仓库
- **固定宽度文本** — 空格对齐，适合日志和纯文本阅读

### 纯文本 → Excel

将纯文本表格转换为 Excel 文件（.xlsx），自动识别：

- **格式** — CSV / TSV / Markdown 表格 / 固定宽度文本
- **数据类型** — 数值（整数、小数、百分数、千分位、科学计数） / 日期（多种格式） / 文本

## 快速开始

### 方式一：直接打开

双击 `表格转换工具.html`，在浏览器中打开。

### 方式二：Docker 部署

```bash
# 构建镜像
docker build -t table-converter .

# 运行容器
docker run -d -p 8080:80 table-converter

# 浏览器访问 http://localhost:8080
```

### 方式三：任意 HTTP 服务器

```bash
# Python 3
python -m http.server 8080

# Node.js (需要安装 http-server)
npx http-server -p 8080
```

## 使用说明

### 功能一：表格转纯文本

1. 将 Excel 或 CSV 文件拖入左侧区域（或点击选择）
2. 在三个格式按钮中选择输出格式：CSV / Markdown 表格 / 固定宽度文本
3. 点击「转换并下载」
4. 文件自动下载到浏览器默认下载目录

### 功能二：纯文本转 Excel

1. 将文本文件拖入右侧区域（或点击选择）
2. 系统自动识别文件格式和每列数据类型，展示在界面上
3. 确认无误后点击「转换并下载 Excel」
4. 生成 .xlsx 文件并自动下载

## 格式检测规则

| 格式 | 特征 | 示例 |
|------|------|------|
| CSV | 逗号 `,` 分隔，每行字段数一致 | `姓名,年龄,部门` |
| TSV | Tab 键分隔，每行字段数一致 | `姓名	年龄	部门` |
| Markdown 表格 | 包含 `\|` 和分隔行 `\|---\|` | `\| 姓名 \| 年龄 \|` |
| 固定宽度 | 连续 2+ 空格对齐，多行位置一致 | `姓名    年龄    部门` |

## 文件编码

- 优先使用 **UTF-8** 编码读取
- 如果检测到乱码，自动尝试 **GBK** 编码
- 输出文件统一使用 UTF-8 with BOM（兼容 Excel 打开中文 CSV）

## 隐私与安全

- 所有文件处理在浏览器本地完成
- **不会**将文件上传到任何远程服务器
- 断开网络后仍可使用（SheetJS 库首次加载后由浏览器缓存）
